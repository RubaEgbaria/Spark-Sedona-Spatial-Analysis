# Spark-Sedona-Spatial-Analysis
Geospatial analytics pipeline using Spark and Sedona to explore business distributions across urban areas. Includes data cleaning, point geometry generation, competitor-distance calculations, and GeoJSON export for mapping. Designed to support spatial decision making and market opportunity analysis at scale.
